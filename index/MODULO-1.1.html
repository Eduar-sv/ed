Aspectos generales.
Campo: Desarrollo de Software
Opcion Desarrollo de Software
Competencia Resolver problemas de lógica computacional utilizando diagramas de flujo y pseudocódigo.
Titulo del modulo Elaboración de algoritmos usando lógica de programación.
Duración prevista Seis semanas (108 horas).
Prerrequesito: 9° grado Código: BTVDS 1.1
Objetivo del módulo:

Desarrollar las competencias para realizar el análisis de sistemas por medio de diagramas de flujo o pseudocódigo para mecanizar un proceso.
Situación problemática:

Situación problemática: Aplicaciones ineficientes de software debido a la mala estructuración lógica en la elaboración del algoritmo que describe el proceso causa un consumo excesivo de recursos informáticos e ineficiencia en la generación de la información.
