ASPECTOS GENERALES
Campo: Desarrollo de Software
Opcion Desarrollo de Software
Competencia Analizar e incorporar requerimientos de un sistema en una propuesta de solución tecnológica realizable.
Titulo del módulo Identificación de requerimientos para diseñar o modificar sistemas informáticos.
Duración prevista Cinco semanas (90 horas).
Prerrequesito: 9° grado Código: BTVDS 1.2
Objetivo del módulo:

El contar con sistemas informáticos que no satisfacen las necesidades del cliente y carentes de elementos de seguridad se debe la falta de compresión de los requerimientos para diseñar sistemas informáticos, condición esencial en la construcción del software; incrementa los costos de implementación y el nivel de insatisfacción del cliente.
Situación problemática:

El contar con sistemas informático que no satisfacen las necesidades del cliente y no cuenta con elementos de seguridad dada la falta de compresión de los requerimientos para la construcción del software se incrementan los costos de implementación y el nivel de insatisfacción del cliente.
