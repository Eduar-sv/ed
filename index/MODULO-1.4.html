ASPECTOS GENERALES
Campo: Desarrollo de Software
Opcion Desarrollo de Software
Competencia Elaborar manuales de sistemas de calidad para estandarizar el desarrollo de software.
Titulo del modulo Elaboración de manual de sistemas de calidad para el desarrollo de software.
Duración prevista Seis semanas (108 horas).
Prerrequesito: 9° grado Código: BTVDS 1.4
Objetivo del módulo:

Desarrollar las competencias para elaborar manuales de sistemas de calidad en el desarrollo de software, utilizando normas y estándares establecidos.
Situación problemática:

A menudo se pueden encontrar aplicaciones cerradas y de baja calidad debido a la poca importancia en la creación de manuales donde se definan los estándares para el desarrollo de software, lo cual ocasiona incremento en tiempo y costo de desarrollo y mantenimiento de los sistemas.
